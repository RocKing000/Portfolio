# How Alignment Intelligence Works

Alignment Intelligence operates through a three-stage process: Reference Manifest definition, portfolio company assessment, and comparative analysis.

---

## Stage 1: Reference Manifest Definition

The institutional client works with the triNetra Research Studio to define a Reference Manifest encoding its investment criteria and structural expectations. This is done once per investment thesis or portfolio segment.

The Reference Manifest is retained by triNetra and applied to every subsequent portfolio company assessment in the engagement. It can be versioned as the investment thesis evolves.

See [reference-manifest.md](reference-manifest.md) for a full explanation of what the Reference Manifest is and how it is structured.

---

## Stage 2: Portfolio Company Assessment

Each portfolio company submits an Input Manifest describing their system: its deployment domain, decision space, structural characteristics, oversight configuration, and traceability posture.

The Eagle Framework (EF) assessment engine evaluates the Input Manifest independently, producing an Eagle Score, maturity level classification, and full Assessment Report for the portfolio company. This is Organisation Intelligence.

---

## Stage 3: Alignment Analysis

The Eagle Framework then compares the portfolio company's assessed structural profile against the Reference Manifest. This produces:

- An Alignment Score
- A Gap Analysis mapping specific structural gaps to specific Reference Manifest criteria
- A Structural Divergence Analysis identifying where the company's architecture conflicts with the institutional standard

---

## Per-Cycle Cadence

Alignment Intelligence operates on a cycle basis. Each Insight Cycle covers a defined assessment period. Portfolio companies are assessed within the cycle, and the comparative deliverables are produced at cycle close.

The Reference Manifest remains constant across cycles unless the institutional client updates it. Updated versions are versioned and dated so that cross-cycle comparisons remain interpretable.

---

## What It Does Not Do

Alignment Intelligence does not implement changes, recommend vendors, or engage directly with portfolio companies. It produces a structural diagnosis identifying where alignment exists and where it does not. The institution receives a governed basis for its own portfolio engagement.
