## What's Here

### 🔴 DIRE-X — Live Product
**Dynamic Intelligence & Risk Engine**  
A gamified geopolitics and supply chain simulation built to solve Smart India Hackathon 2025's problem statement: *"Gamifying the learning experience for kids."*

**Live:** [dire-x.vercel.app](https://dire-x.vercel.app)

Built solo. Full-stack: React/Vite (Vercel) · Node.js game server (Render) · Python FastAPI scoring engine (Render) · PostgreSQL (Supabase).  
Models 92 countries · 80+ companies · 15 strategic resources · 7 crisis types · 84 API endpoints · 20 simulation engines · ~15,000 lines of code.

Core differentiator: **SRES (Strategic Resource Evaluation Score)** — a proprietary multi-dimensional risk index giving each company a live, resource-weighted score across supply, demand, geopolitical, and environmental pressures simultaneously.

Age-adaptive UI across 3 tiers: Explorer (13–17) · Strategist (18–24) · Analyst (25+)

*Origin: I couldn't visualise academic concepts as a teenager. This is the tool I needed then.*

## About the AI Usage

Two distinct uses of generative AI across this work — kept deliberately separate:

**1. As a product component**  
OpenRouter (Claude/GPT) is integrated directly into DIRE-X for AI narration and strategic insights within the simulation.

**2. As a research stress-tester**  
ChatGPT and Claude were used throughout the research and framework work in this repository — not to generate content, but to probe where AI oversimplifies, hallucinates domain knowledge, or loses cultural and geopolitical nuance. Both uses inform how I think about AI's role in products.
